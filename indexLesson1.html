Урок 1. Основы языка JavaScript

1. Задать температуру в градусах по Цельсию. Вывести в alert соответствующую температуру в градусах по Фаренгейту.
Подсказка: расчёт идёт по формуле: Tf = (9 / 5) * Tc + 32, где Tf – температура по Фаренгейту, Tc – температура по Цельсию


var tC = prompt("Введите температуру по Цельсию:");
var tF = (9 / 5) * tC + 32;
alert("Температура по Фаренгейту: " + tF);

5. *Самостоятельно разобраться с атрибутами тега script (async и defer)

async - загрузка скрипта идет параллельно с загрузкой страницы. После того как скрипт загрузился, HTML парсинг страницы приостанавливается, выполняется скрипт JS, и после того как скрипт выполнен, продолжается загрузка страницы. Скрипты с атрибутом async не зависят друг от друга, порядок загрузки всегда разный и запускается тот, который быстрее всех загрузился.
defer - загрузка скрипта идет параллельно и парсинг страницы не блокируется. Скрипт начнет выполнятся только после того как все теги html будут загружены на страницу. Для максимальной производительности, целесообразно данный скрипт подключать в начале тега head. Скрипты выполняются строго в том порядке, в котором они расположены.
